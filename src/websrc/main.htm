Windows Software
A Windows monitoring software is available for download.


The easiest way to install the software is to use the one click deploy feature (MS). You will stay up to date and get the software immediately. Microsoft uses an executable file, with the name "setup.exe". From security point of view, this is not a good idea, but it is the easiest way to install the software.
To limit the risk, the software is signed with a certificate from LSC-Labs. If you choose this way and you want to avoid a lot of warnings, install the root and the intermediate certificates from LSC-Labs first (be sure you are installing it from the correct page "https://lsc-labs.de/Certificates").

Download the software (zip package)
Unpack the software into a folder you like (i.E. C:\Apps\OnAirNotifier). As it is a portable version, you can start the software from any location (also from a USB stick) by starting the OnAirNotifier.exe.
Be aware, that you have also a security risk, so ensure you are on the right download page!
